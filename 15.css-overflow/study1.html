overflow

overflow - 흘러넘친 컨텐츠에 대해서 대응하기위한 css 속성이다

  • hidden - 흘러넘친 컨텐츠에 대해서 안보이게 처리
  • scroll - 흘러넘친 컨텐츠에 대해서 스크롤바를 표현
  • auto - 흘러넘친 컨텐츠에 대해서 스크롤바를 표현 (텍스트가 흘러 넘치면 스크롤 표시, 흘러넘치지 않으면 스크롤를 표시하지 않는다)
  • visible (기본값) - 흘러넘친 컨텐츠를 그대로 보여주도록 처리
